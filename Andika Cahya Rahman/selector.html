Kesimpulan

Hasil Fungsi :

Kesimpulannya untuk menunjukkan berbagai penggunaan pseucode dan selector di dalam html mulai dari memodifikasi daftar list hingga mengubah warna tombol dan background pada tombol.

  1. Terdapat 3 Daftar Produk yang berisi 3 produk,yaitu Laptop,Smartphone,Dan Tablet dengan menggunakan li
    Kegunaan : Memberi list pada Daftar produk

  2. Jika cursor di arahkan ke tombol "Beli Sekarang",maka background dari tombol tersebut akan berubah warna dan warnanya berubah jadi warna putih
    Kegunaan : Mengubah warna dan background pada tombol saat cursor diarahkan dan di tekan dengan menggunakan button:hover

  3. Jika list tersebut berada di urutan ke 2,maka list tersebut akan berubah warna menjadi warna biru dengan menggunakan li:nth-child(2)
    Kegunaan : Efek ini membantu menyoroti item tertentu berdasarkan urutannya dalam daftar

  4. Tombol "Link Nonaktif" digunakan untuk menampilkan link atau tombol yang terlihat, tetapi tidak dapat diklik atau tidak bisa diakses oleh pengguna.Dengan menggunakan a[class="disabled"]
    Kegunaan : Untuk Menonaktifkan tautan/aksi pada tombol.
